# **App Name**: ScholaDesk AI

## Core Features:

- Secure User Authentication & Authorization: Enable secure login for students and professors with distinct role-based access controls to different features and content.
- Resource Hub: Professors can upload, organize, and categorize learning materials (documents, videos, links). Students can browse, search, view, and download these resources.
- Student Profile & Grade Management: Professors input and update student grades. Students can securely view their individual academic performance and progress reports.
- AI-Powered Exercise Suggestions: A generative AI tool analyzes individual student grades and performance data to suggest personalized exercises, practice problems, and study materials.
- Course & Class Administration: Professors can create, edit, and manage courses, assign students to their classes, and oversee their roster and course-specific settings.
- Personalized Dashboards & Notifications: Provide a personalized dashboard for each user role (students get upcoming assignments and AI suggestions; professors get class overviews and activity alerts).

## Style Guidelines:

- Primary color: A deep, professional blue (#1F71AD) to convey trust, clarity, and education.
- Background color: A very light, soft blue-grey (#F0F2F5) for a clean, spacious, and modern aesthetic, ensuring high readability.
- Accent color: A vibrant yet sophisticated teal (#26D9BA) for interactive elements and key highlights, offering a fresh, innovative feel.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable design suitable for both headings and extensive content blocks.
- Utilize minimalist, crisp line-art icons to complement the modern and professional visual design.
- Adopt a clean, grid-based, and responsive layout with clear visual hierarchy and generous use of white space to enhance focus and usability.
- Incorporate subtle animations for navigation transitions, element hovers, and data updates to provide a polished and engaging user experience.
